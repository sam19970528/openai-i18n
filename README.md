# Open AI 多國語言翻譯網

使用 Open AI API 的 chat 模型製作出的多國語翻譯工具

## 安裝方法

要安裝此專案，請遵循以下步驟：

1. 確保你的電腦已經安裝了 Node.js 環境。如果還沒有安裝，你可以在[官方網站](https://nodejs.org)下載並安裝。

2. 使用終端機進入專案的根目錄。

3. 在終端機中運行以下命令，以安裝專案所需的相依套件：

```shell
npm install

```

## 啟動專案

要在本地環境運行此專案，請執行以下步驟：

1. 在終端機中運行以下命令，以啟動本地開發伺服器:

```shell
npm run dev

```
